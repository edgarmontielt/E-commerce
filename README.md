## Getting Started

First, rebuild node_modules:

```bash
npm install
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
```

For production server

```bash
npm run start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
